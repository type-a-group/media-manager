Connect Google Photos — Setup

✕

One-time setup. You’ll use your own Google credentials, so nothing is shared with anyone.

  1. Open console.cloud.google.com → create a project
  2. Enable the Google Photos Picker API
  3. Create an OAuth client → type Desktop app
  4. Add your own email as a test user
  5. Paste the credentials below ↓
ℹ
Stored locally in media/google.json (chmod 600). Never uploaded, never committed.
